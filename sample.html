Sample Excolor Useage

Useage should be simple enough, just:

  1. Link to the jquery library however you like ( tested with 1.6.4 )

    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js" type="text/javascript"></script>

  2. Then, link to the excolor library.

    <script src="excolor.js" type="text/javascript"></script>

  3. And finally, in your code you should call excolor on the entities you want to take color input like this:

    $('input.color-input').excolor();
